# Learnable Meta integration notice

This OhneGuessr plugin interoperates with the public userscript API provided by
[Learnable Meta](https://learnablemeta.com/).

Parts of the code in this directory are adapted from the
[Learnable Meta userscript](https://github.com/likeon/geometa), including parts
of its clue image carousel and magnifier behavior. The adapted code was modified
for OhneGuessr in July 2026.
